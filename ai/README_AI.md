# In-Process Setup (Claude Owner)

## Agent Teams (in-process)
Agent Teams sind aktiviert und laufen in-process.
- Claude zeigt Teams im gleichen Terminal.
- Wechseln zwischen Teammates: **Shift+Down**.

## Nutzung
- Agents anzeigen: `/show-agents`
- Kleiner Task: `/do <task>` oder Prompt mit `@do <task>`
- Großer Task: `/big <task>` oder Prompt mit `@big <task>`

Outputs:
- `.ai/dispatch/latest/` enthält SUMMARY.md, SMOKE.txt, I18N.txt, DIFFSTAT.txt
